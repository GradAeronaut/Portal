# ИНСТРУКЦИЯ: Установка темы Swift для XenForo

## ⚠️ ВАЖНО
Выполните эти шаги **в админ-панели XenForo** через веб-интерфейс.

---

## Этап 1 — Импорт темы (без активации)

1. Зайти в админ-панель XenForo
2. Перейти: **Appearance → Styles → Import**
3. Импортировать тему из файла:
   ```
   /forum/_vendor/themes/swift/swift_v237.zip
   ```
4. Убедиться, что стиль **Swift** появился в списке стилей
5. **НЕ назначать** стиль Swift стилем по умолчанию

---

## Этап 2 — Создание child-style

6. Перейти: **Appearance → Styles → Add style**
7. Создать новый стиль со следующими параметрами:
   - **Name:** `Swift – Sinbad`
   - **Parent style:** `Swift`
8. Сохранить стиль

---

## Этап 3 — Минимальная первичная настройка (строго без кода)

9. Открыть: **Swift – Sinbad → Style properties**
10. Выполнить только следующие действия (если опции существуют):
    - ✅ **Отключить background images:**
      - node background
      - header background
      - page background
    - ✅ **Отключить или установить в `none` gradients**
    - ✅ **Выбрать один фиксированный color preset** (без переключателей)
    - ✅ **Снизить визуальный контраст:**
      - secondary text
      - borders
      - dividers
    - ✅ **Выбрать нейтральный системный шрифт:**
      - sans-serif
      - без декоративных шрифтов

**⚠️ ЗАПРЕЩЕНО:**
- ❌ редактировать шаблоны
- ❌ добавлять Custom CSS / LESS
- ❌ менять HTML-структуру
- ❌ устанавливать дополнительные аддоны

---

## Этап 4 — Активация

12. Назначить стиль **Swift – Sinbad** стилем по умолчанию для форума
13. Очистить кэш XenForo:
    - **Tools → Rebuild caches** или
    - **Tools → Cache → Clear cache**

---

## Git: Фиксация изменений

14. Если в процессе были затронуты файлы проекта:

```bash
cd /Users/user/SinbadRepo/sinbad-portal-local
git add .
git commit -m "Install Swift theme and create Sinbad child style"
git push
```

**Примечание:** Если установка темы выполнялась только через админ-панель XenForo, файлы проекта могут не измениться, так как тема хранится в базе данных XenForo.

---

## Проверка после установки

После выполнения всех этапов проверьте:

✅ Стиль **Swift – Sinbad** отображается в списке стилей  
✅ Стиль **Swift – Sinbad** назначен стилем по умолчанию  
✅ Форум отображается с новым стилем  
✅ Внешний вид соответствует требованиям (без декоративных элементов, нейтральный)  
✅ Кэш XenForo очищен  

---

## Откат изменений (если необходимо)

Если нужно вернуться к предыдущему стилю:

1. Перейти: **Appearance → Styles**
2. Назначить предыдущий стиль стилем по умолчанию
3. Очистить кэш XenForo

---

## Примечания

- Файл темы находится по пути: `/forum/_vendor/themes/swift/swift_v237.zip`
- Child-style наследует все настройки от родительского стиля Swift
- Изменения в child-style не затрагивают оригинальную тему Swift
- Все настройки можно изменить позже через **Style properties**
